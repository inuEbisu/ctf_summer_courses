## Task 1 (30%)

课堂中提及到了，从源代码到汇编可以使用不同的编译器: `gcc` 与 `clang`；
同时，从汇编到目标文件/可执行文件，也可以使用不同的汇编器: `as` 与 `llvm-mc`；

那么，请混搭使用，探究 `gcc` + `llvm-mc` 与 `clang` + `as` 来完成程序的编译与汇编，程序不限，可以使用简单的 `hello.c`

请在报告中截图给出混用效果，混用都可以成功么？如果失败，请尝试分析原因
